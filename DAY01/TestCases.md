# Login Test Cases

## Positive Test Cases

- Login with valid email and password
- Redirect to Dashboard after successful login

## Negative Test Cases

- Invalid email
- Invalid password
- Both invalid
- Empty email
- Empty password

## Validation Test Cases

- Verify email format
- Verify mandatory fields
- Verify Login button state

## Boundary Test Cases

- Maximum email length
- Minimum password length
- Special characters
