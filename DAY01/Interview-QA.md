# Interview Questions

## 1. What is AI?

Artificial Intelligence enables machines to perform tasks requiring human intelligence.

---

## 2. AI vs ML vs DL

AI → Broad concept

ML → Learns from data

DL → Uses deep neural networks

---

## 3. What is Generative AI?

Generates new content like text, code, images, and documents.

---

## 4. What is an LLM?

A Large Language Model trained on massive text data.

---

## 5. What are Tokens?

Small units of text processed by an LLM.

---

## 6. What is a Context Window?

Maximum number of tokens processed in one interaction.

---

## 7. What is Temperature?

Controls creativity of AI responses.

---

## 8. What are Hallucinations?

Incorrect or fabricated AI responses.

---

## 9. How does AI help QA Engineers?

- Generate test cases
- Generate automation scripts
- Create API payloads
- Debug code
- Analyze errors
- Create SQL queries

---

## 10. Why should AI-generated code be reviewed?

Because AI can produce incorrect or outdated code, so it should always be validated.
